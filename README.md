# 🚗 Prawko Jazdy

Aplikacja webowa do nauki i przygotowania do egzaminu na prawo jazdy. Projekt składa się z frontendu (JavaScript), backendu (Node.js) oraz skryptów pomocniczych w Pythonie.

## Struktura projektu

```
prawko-jazdy/
├── frontend/       # Interfejs użytkownika (JavaScript, CSS, HTML)
├── backend/        # Serwer API (Node.js)
└── Backend Py/     # Skrypty pomocnicze (Python)
```

## Wymagania

- Node.js
- Python 3.x
- npm

## Uruchomienie

### Backend (Node.js)

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Skrypty Python

```bash
cd "Backend Py"
pip install -r requirements.txt
python main.py
```

## Technologie

- **Frontend:** React.js
- **Backend:** Node.js
- **Skrypty:** Python
